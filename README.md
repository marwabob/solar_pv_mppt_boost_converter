# Boost Converter with MPPT (P&O) in MATLAB Simulink

This project presents the design and simulation of a DC-DC Boost Converter integrated with the Perturb and Observe (P&O) MPPT algorithm for a 4kW Solar PV system.

## Features
- Boost converter design in MATLAB Simulink
- MPPT using Perturb & Observe algorithm
- 4kW solar PV system simulation
- Output voltage, current, and power analysis

## Tools Used
- MATLAB
- Simulink

## Simulation Results
The system successfully tracks the maximum power point under varying conditions and maintains stable output.

## How to Use
1. Open the `.slx` file in MATLAB Simulink
2. Run the simulation
3. Observe output waveforms (Voltage, Current, Power)
